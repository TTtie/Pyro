# Pyro-Bot
Bring Pyro from TF2 experience to your Discord server

- [Node.js](https://nodejs.org)
- Eris; `npm install eris`
- FFmpeg; get it yourself, not providing binaries
- colors; idk why i did it; `npm install colors`
